Experience

Garmin Software Engineer Intern

As a Software Engineer Intern at Garmin in Chandler, AZ, I developed safety-critical embedded C software for multiple generations of Garmin Integrated Flight Decks, ensuring compliance with industry standards and regulations. I conducted thorough testing by creating an automated regression testing suite in Python. With bugs being identified I worked to resolve them through software and requirement changes. Additionally, I implemented customer-requested features to enhance the functionality of the flight decks and leveraged the software architecture to enable runtime-determined features. Throughout the development process, I utilized Git to manage and track changes, ensuring a streamlined workflow.

The University of Arizona SIE Disruption Lab Student Software Developer

During my time as a Student Software Developer at the University of Arizona, I developed a full-stack web application for the Systems and Industrial Engineering Disruption Lab. This application converted and visualized Bayesian networks from OML, significantly enhancing accessibility for researchers. I conducted thorough testing and debugging, achieving a 95% reduction in reported bugs during the testing phase. The project included developing a Flask backend with RESTful APIs and containerizing the app with Docker for seamless deployment. I also created a responsive frontend using HTML, CSS, and JavaScript, which improved usability and user satisfaction by 50%.

NASA Jet Propulsion Laboratory Software Engineer Intern

At NASA's Jet Propulsion Laboratory in Pasadena, CA, I served as a Software Engineer Intern, where I developed and optimized Python scripts using open-source libraries, Jama, and TestRail APIs, improving runtime by 32%. I also led a team of interns to develop an open-source software solution for an internal application migration. My responsibilities included implementing CI/CD pipelines with Jenkins for automated testing and deployment of pip packages, managing workload through Agile and Scrum methodologies, and effectively handling tickets and metrics across multiple sprints using Jira. I conducted internal customer-facing interactions to communicate project updates and resolve concerns, while utilizing Git and GitHub for version control and team collaboration.
