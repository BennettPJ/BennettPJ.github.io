FPGA Processor Design

Implemented a FPGA-based full scale pipelined datapath with tradeoff analysis on the joint effects of the hardware, software, and instruction set architecture design realms. I designed, implemented, and validated a five-stage pipelined datapath for the MIPS 32-bit ISA on the Xilinx Artix-7 FPGA. Additionally, I developed a video processing algorithm in MIPS ISA and executed it on the FPGA-based emulation of the pipelined processor.

Spotify Weekly Playlist Saver

The aim of this project was to create a practical tool for my daily use. Spotify curates weekly playlists for users, but these playlists are not saved from week to week. I wanted to preserve these songs since I often missed the chance to listen to them before they disappeared. To solve this, I wrote a Python script that utilizes Spotify's API to save the 30 curated songs to a permanent playlist each week. Additionally, I implemented GitHub Actions to automate the script's execution on a weekly basis. I found this project particularly rewarding as it provided a useful solution that I could integrate into my everyday life.
